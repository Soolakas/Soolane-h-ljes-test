# Plan: Balance Changes + Developer Mode

## Balance Changes

### 1. Enemy Spawn Distance
- **Problem**: Enemies can spawn very close to the player, causing unfair deaths
- **Fix**: Add `MIN_SPAWN_DISTANCE_FROM_PLAYER` = 250px (same as Heavy Metal radius) in spawn_manager.py
- **Logic**: When selecting spawn point, check distance to player. If too close, pick new point or push spawn point further away along the spawn ray

### 2. Thruster Tuning
- **Change**: +8% → **+10%** speed per stack
- **File**: `upgrade_registry.py` - change `+0.08` to `+0.10`

### 3. GPS Tracker
- **Change**: Perfect accuracy at **+2 stacks** instead of +3
- **Formula**: `effective_spread = base_spread * max(0, 1.0 - accuracy * 0.5)` (at +2, spread = 0)
- **File**: `game.py` shooting section

### 4. Green Juice (Buff Stacking)
- **Change**: Increase stacking chance and damage
- **New formula**:
  - 1st: 15% chance, 0.5 HP/s
  - 2nd: +10% chance (25% total), +0.15 HP/s (0.65 HP/s total)
  - 3rd: +7% chance (32% total), +0.10 HP/s (0.75 HP/s total)
  - Formula: `chance_increase = 0.10 * 0.7^(stacks-2)` for stacks >= 2
  - Formula: `damage_increase = 0.15 * 0.67^(stacks-2)` for stacks >= 2
- **File**: `upgrade_registry.py` `_green_juice_apply()`

### 5. Waller (Buff Bounce Velocity)
- **Change**: 2nd stack gives +40% velocity on bounce, incremental increase, cap at 250%
- **New formula**: `min(2.5, 1.0 + 0.4 * max(0, stacks - 1))`
  - Stack 1: 1.0x (100%)
  - Stack 2: 1.4x (140%)
  - Stack 3: 1.8x (180%)
  - Stack 4: 2.2x (220%)
  - Stack 5+: 2.5x (250% cap)
- **File**: `upgrade_registry.py` `_waller_apply()`

### 6. The Shift Key (Shorter Dash)
- **Change**: Dash duration halved: 0.15s → **0.075s**
- **File**: `game.py` dash_state["duration"]

### 7. Heavy Metal (Check + Buff)
- **Current status**: Works but `int()` truncation makes small bonuses ineffective
- **Solution**: Increase base damage from 1 → **10**, scale all related values ×10:
  - Enemy base HP: ×10 (common: 2→20, fast: 1→10, tank: 5→50)
  - Difficulty health bonus: ×10 (MAX_HEALTH_BONUS: 8→80)
  - Poison damage: ×10 (0.5→5.0 HP/s base)
  - Cactus armor damage: ×10
  - Use `round()` instead of `int()` for damage calculation
- **Change**: 
  - Base: **+50%** damage (was 20%)
  - Further stacks: **+20%** each
- **Files**: `game.py` (base damage, enemy.py (base HP), difficulty_manager.py, upgrade_registry.py

### 8. Enemy Speed Cap
- **Change**: Max speed multiplier from 3x → **2x**
- **File**: `difficulty_manager.py` `MAX_SPEED_MULTIPLIER = 2.0`

## Developer Mode

### Activation
- Press **"+"** key to toggle developer mode on/off
- Only available during PLAYING state
- Kept in final build (toggleable)

### Developer Panel
- Semi-transparent overlay panel in top-right corner
- Contains clickable buttons/options:

#### Options:
1. **Toggle Invincibility** - Toggle player invincibility on/off (visual indicator when active)
2. **Grant Upgrade** - Opens submenu showing ALL upgrades organized by rarity (including owned). Click any to instantly apply.
3. **Show Info** - Toggle debug info overlay showing:
   - Player stats (all current values)
   - Current difficulty scaling values (spawn interval, wave size, health bonus, speed multiplier, type weights)
   - Current time in run
   - Number of active enemies
   - Current upgrade stacks
4. **Skip Time** - Add 30s / 60s / 5min to elapsed time (for testing scaling)
5. **Kill All Enemies** - Instantly clear the board
6. **Reset Run** - Restart the current run without going to menu

### Info Overlay Format
